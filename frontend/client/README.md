# GoNotes Client

This is the client AHHH

## Running the Application

1. Navigate to : `GoNotes/frontend/client`
2. Install dependencies: `flutter pub get`
3. Launch an emulator:
   - Press `Ctrl + Shift + P`
   - Search for and select **Flutter: Launch Emulator**
   - Choose/create an Android emulator
4. Run the app in the same directory: `flutter run`

## Features
- Display Google Maps with initial location in Copenhagen
- Request and manage location permissions
- Can post a note to the backend (run backend via `backend\README.md`)
