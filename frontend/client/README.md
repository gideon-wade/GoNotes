## Notes
If emulator gets stuck on boot you can wipe the data of it by typing:

```
& "$env:LOCALAPPDATA\Android\Sdk\emulator\emulator.exe" -avd flutter_emulator -wipe-data
```

Where `flutter_emulator` is the emulator you want to wipe.
